Cody's Unity Adventures


Cody's Unity Adventures is a game about learning Unity. You play the game in the Unity Editor and solve the missions you are given to unlock the goal.






How to play

Installation

To start playing Cody's Unity Adventures you need to do the following steps.

You can download the Git Bash, so you can use the Unity Editor navigation to move between Mission since each mission is on their own branch. This requires the PATH setup for git too (This might require a PC reboot after installing Git). Or you can use Git Hub Desktop and manually change branch from the dropdown menu.

Copy the git url and use it to clone the project on your local drive.

You can manually download the correct version from Unity archives, or you can try to open the cloned project in Unity Hub and it suggest downloading the correct version automatically. The Unity version is marked with a know security issue just like most of them are before 6.2.1f, but it's not an issue since this game is not a "release" / build

Mission

Each mission has their own objective you need to complete to activate the Trophy (goal).
There are regular missions, quizes, and boss fights that have multiple issues to fix.

You can see the mission on each level by looking at the Game view, or starting the play mode (which technically takes you to the game view as well)

Quiz

Every 4th level is a Quiz. And to complete the quiz you need to answer 5/10 questions correctly. If you fail to answer 5/10 correctly just start the play mode again.

Boss "Fight"

In every boss fight your project is devastated by two prankster cats. To reach the goal you have to fix everything the cats have broken and remove their pranks to reach the goal.





Hints

Here you can find 3 hints for each mission. After you open a hint, the next Hint-button becomes visible.
